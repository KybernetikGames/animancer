Adds the `state` to this selector with its Animancer.FSM.IPrioritizable.Priority.
CanSetState() bool
Calls Animancer.FSM.StateMachine`1.CanSetState(`0) for each state in this selector in order of priority (highest first) until one returns true. Otherwise this method returns false.
Clear() void
Removes all states from this selector.
ForceSetState() bool
Calls Animancer.FSM.StateMachine`1.ForceSetState(`0) for the highest priority state in this selector and returns true. Or does nothing and returns false if there are no states.
TryResetState() bool
Calls Animancer.FSM.StateMachine`1.TryResetState(`0) for each state in this selector in order of priority (highest first). If one of them returns true, this method calls Animancer.FSM.StateMachine`1.StateSelector.Clear and returns true. Otherwise it returns false.
TrySetState() bool
Calls Animancer.FSM.StateMachine`1.TrySetState(`0) for each state in this selector in order of priority (highest first). If one of them returns true, this method calls Animancer.FSM.StateMachine`1.StateSelector.Clear and returns true. Otherwise it returns false.
