Serialization.

ResetValue(SerializedProperty, string) Method

Summary

Resets the value of the UnityEditor.SerializedProperty to the default value of its type and all its field types, ignoring values set by constructors or field initializers.
Assembly
Animancer.dll
Namespace
Animancer.Editor
Containing Type
Serialization

Syntax

public static void ResetValue(SerializedProperty property, string undoName = "Inspector")

Remarks

If you want to run constructors and field initializers, you can call Animancer.Editor.Serialization.PropertyAccessor.ResetValue(UnityEditor.SerializedProperty,System.String) instead.
