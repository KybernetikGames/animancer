[Editor-Only] Draws the Inspector GUI for an Animancer.ITransition.

Syntax

[CustomPropertyDrawer(typeof(PlayableAssetTransition), true)]
public class PlayableAssetTransitionDrawer : TransitionDrawer, IPolymorphic

Remarks

Implement this interface in a UnityEditor.PropertyDrawer to indicate that it should entirely replace the Animancer.Editor.PolymorphicDrawer.
