Animancer is a Unity plugin which allows you to play animations on demand and control their details directly, without all the restrictions and extra preparation required by other systems.

Introduction

Compare the differences between Unity's animation systems.

Download

Try out all the features in the Unity Editor for FREE with Animancer Lite.

Quick Play

Just get an animation, play it, and directly control its details in your scripts.

Examples

Extensively documented example scenes demonstrate how to use Animancer.

Performance

Animancer can outperform other systems, but the difference is fairly small.

Integration

Use Animancer alongside an existing Animator Controller based setup.

State Machine

Code based Finite State Machine is entirely separate from the animation system.

Live Inspector

View real-time animation details and manual controls for debugging and testing.

Transitions

Preview animations and tweak their details in the Inspector.

Fading

Smoothly blend between animations over time.

Mixers

Arbitrarily blend multiple synchronised animations.

Layers

Play separate animations on different body parts.

Events

Use Unity's Animation Events or Animancer's custom event system for more flexibility.

Directional Animations

Easily manage groups of
up/down/left/right animations.

Inverse Kinematics

Use Unity's inbuilt IK system for Humanoid models.

Editor Tools

Pack textures, mass-modify sprites, generate sprite animations, and more.

Animation Jobs

Use custom scripts to dynamically modify the animation stream at runtime.
